Pip安装

2016-12-25

下载源代码

  • https://pypi.python.org/pypi/pip 我去下载的时候是 8.0.2版本

解压

  • tar xvzf pip8.0.2.tar.gz

安装

  • cd pip-1.4.1
  • sudo python setup.py install
  • pip install numpy
  • pip install lxml
  • sudo pip install requests
  • sudo pip install openpyxl

中文乱码解决

  • SyntaxError: Non-ASCII character ‘\xe7’ in file setup.py
  • 在脚本第一天行加上如下内容
    • #!/usr/bin/python
    • #coding:utf-8
